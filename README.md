# Vermont Ski Tracker

Track your ski days at Vermont and New England mountains with your crew!

## Features
- Track ski days for multiple users (Aiden, Jack, Matt, Mike, Reece)
- Record conditions, snowfall, temperature, favorite runs
- Mobile-optimized version that works like an app
- Vermont hippy aesthetic with Galaxy BT font
- Data syncs between mobile and desktop

## Quick Start

### For Users:
1. **Desktop**: Visit the website on your computer
2. **Mobile**: 
   - Open the site in Safari (iPhone) or Chrome (Android)
   - Tap Share → "Add to Home Screen"
   - It will install like a regular app!

### For Setup:
All files needed are in this folder:
- `index.html` - Auto-detects mobile/desktop
- `ski-tracker.html` - Desktop version
- `ski-tracker-mobile.html` - Mobile version
- `styles.css` & `styles-mobile.css` - Styling
- `script.js` & `script-mobile.js` - Functionality
- `manifest.json` - PWA configuration
- `802.jpg` & `Steely.webp` - Vermont images

## Data Storage
- Data is stored locally on each device
- Each user's ski days are kept separate
- Data persists between sessions

## Hosting on GitHub Pages

1. Create a new GitHub repository
2. Upload all files from this folder
3. Go to Settings → Pages
4. Source: Deploy from a branch
5. Branch: main (or master)
6. Folder: / (root)
7. Click Save

Your site will be live at:
`https://[your-username].github.io/[repository-name]`

Share this URL with your buddies and they can add it to their phones!
